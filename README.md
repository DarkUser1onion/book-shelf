# Книжная полка
Простое консольное приложение для учёта прочитанных книг.

## Запуск
python main.py

## Функции
- Добавление книги (название, автор, оценка)
- Просмотр списка
- Удаление книги

## Инструменты разработки

- Форматирование: [Black](https://black.readthedocs.io/)
- Линтер: [Ruff](https://docs.astral.sh/ruff/)
- Pre-commit hooks: автоматическая проверка и исправление кода перед коммитом

Для установки pre-commit выполните:
```
pre-commit install
```
